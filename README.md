# GenZ Studio

An AI-powered creator platform for designing YouTube banners, thumbnails, profile pictures, editing images and videos, and generating AI-powered content—all in one place.

## Prerequisites

- Node.js 22.23.1
- npm
- Git

## Setup

```bash
git clone <repo-url>

cd Genz-studio

nvm use

cd client

npm install

npm start
```

Application runs on:

http://localhost:4200
